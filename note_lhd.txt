# 용어 정리(동기화)
- [셀] : 지뢰찾기의 최소 크기 단위. 지뢰 숫자 깃발등을 포함할수있다. 지뢰셀, 숫자셀, 깃발셀, 빈셀 등 ≒ [칸]
- [칸] : 셀의 유사어. 하나의 크기보다 넓은범위를 부를때 주로 쓰임 ≒ [셀]
- [활성화?] : 셀의 상태 중 클릭이나 선택을 하여 출력된 상태 <-> [비활성화?]
- [지뢰] : 활성화 되면 게임이 패배되는 요소. 말그대로 지뢰 ≒ [마인]
- [숫자] : 자기의 셀 기준으로 팔방에 몇개의 지뢰가 존재하는지 나타내는 요소
- [깃발] : 사용자가 특정한 표시를 하기 위한 요소. 보통 지뢰의 위치에 나타내며 클릭을 할수없다.
- [빈셀] : 숫자나 지뢰, 깃발이 존재하지 않고 그저 비어있는 공간
- [커버?] : 비활성화 되어있는 셀. 어떤 요소가 존재하는지 알지못한다. '요소'에 커버를 붙이면 비활성화된 '요소'를 뜻한다. ex)커버지뢰 : 비활성화 지뢰

- [Combo] 일정 횟수를 틀리지(생명이 있는 맵에서) 않고 플레이 할 경우 랜덤아이템을 지급하거나 점수 배율을 높인다.//생명이 없어도 게이지형식으로 일정 게이지를 채우면 아이템지급을 한다거나..

## 아이템 //아이디어일뿐 실제 구현했을때 재미없거나 구현이 힘들땐 제외
- 아이템 지급 방식 :
  - 지정된 칸을 제거하는데 성공했을때 지급
  - 일정한 개수를 제거하는데 성공했을때 지급
  - 지정된 칸에 코인을 지급하여 직접 구매
  //아이템 지급은 랜덤박스를 사용하거나 처음부터 아이템이 구현되고 랜덤한 위치에 배치되거나 지급한다.
- 아이템 효율에 따라 레어도를 정해서 아이템 확률 구현//확률을 미리 정하면 비교적 간단
		//1성 2성 3성 처럼 등급을 정하고 등급마다 일정 확률을 가지며 나눠진 확률을 다시 같은 등급끼리 동등하게 나누는 방법이 있다.
- 아이템 소지 칸 제한
- (2인 이상 플레이 시)상대방들의 아이템 현황

### 이로운 아이템
- [랜덤박스?] : 랜덤한 아이템 가져옴//랜덤박스 등급에 따라 가져오는 아이템 등급을 결정, 또는 등급마다 확률 조정
- [선택?] : 지정한 아이템을 가져옴//가장 상위 아이템
- [첩자?] : (2인 이상 플레이 시)상대방의 아이템을 가져옴
- [생명?]
- [돋보기?] : 사용할 위치를 지정하여 일정 범위(1칸, 4칸, 9칸 등)를 밝힘 //밝히는 범위에 지뢰가 있을때 깃발로 표시? 강제로 지뢰가 제거되는 형태?
	//임펙트를 돋보기가 아닌 폭발로 밝히는것도 가능할듯.//밝히는 범위를 사각형이 아닌 일자형, 원형, 부정형 등 다르게 표시하는 것도 괜찮아보임.
- [EOD(Explosive Ordnance Disposal)?] : 일정 개수(난이도별 변동. 비율(5%,10%) 또는 고정개수(5,10,15) 제거)만큼 지뢰를 (제거/표시)한다. //개인적으론 제거한거에 한표
- [Destroy] : 게임 맵을 강제로 파괴시켜(가로,세로 n줄) 축소시킨다. 파괴된 지뢰는 카운터에서 줄어든다.
- [확대?] : 지뢰개수를 유지한채 맵의 크기만 늘어남
- [반사?] : (2인 이상 플레이 시)상대방의 아이템 효과를 되돌림
- [실드?] : 해로운 효과 1번 방어
- [공격?] : (2인 이상 플레이 시)방해 아이템 효과를 상대에게 일으킴
- []

### 방해 아이템
- [지뢰매설?] : 지뢰추가적인 지뢰를 생성
- [안개?] : 활성화된 셀을 일정 범위(9,16,25칸) 가린다.//활성화 셀을 다시 비활성화 시킴
- [시간정지?] : 일정시간 클릭을 막는다.
- [50%?] : 50% 찍기로 맞혀야하는 지뢰를 강제로(모서리에 50%지뢰를 세팅후 게임판을 재배치) 활성화시킴. //조금 무리수?//지뢰 개수에 영향을 미쳐서 힘들듯
		//아예 따로 미니사이즈 판을 구현시켜서 구석에 배치시키면?
- [반전?] : 맵 구성을 반전시킨다.
- [지진?] : 맵을 랜덤하게 재배치 시킨다.//재배치 되기전 활성화된 셀의 개수만큼 재배치 후에 셀을 활성화 시켜준다.
- [배신?] : (2인 이상 플레이 시)나의 아이템을 랜덤한 상대에게 줌//아이템 없을시 앞으로 얻을 아이템 한개를 주거나, 랜덤지급하거나, 아무런 효과도 없거나,, 선택
- [축소?] : 지뢰개수는 유지한채 맵의 크기만 줄어듬//전체 셀의 수가 지뢰개수보다 많을시에만
- [Reverse?] : 우클릭 좌클릭을 반대로 적용시킨다.
- [Auto?] : 일정시간 마우스 커서가 지나간 자리를 자동으로 활성화 시킨다.

### 특수 //아이템으로 구현해도 되고 게임시작전에 설정하여 게임이 끝날때까지 효과를 유지하거나 랜덤하게 효과를 발휘하게 만들수있음
- [지정?] : (2인 이상 플레이 시)다음에 사용할 아이템 효과가 랜덤한 유저에게 사용됨
- [랜덤?] : 랜덤한 아이템 효과가 랜덤한 유저에게 사용됨
- [폭탄돌리기?다이너마이트?] : (생명이 있을 시/2인 이상 플레이 시) 랜덤한 유저에게 폭탄을 지급하고, 지급된 폭탄은 게임을 진행하면(셀 활성화) 다음 사람에게 넘어간다.
			랜덤한 턴이나 시간이 지나면 폭발
- [Sudden Death?] : (생명이 있을 시) 일정한 시간마다 생명 소모
- [Virus?] : 중앙에서 생성. 맵을 느린속도로 (랜덤하게/비활성화 칸을 향해) 돌아다님. 지나간 자리를 활성화시킴.
		지뢰는 터지고 깃발은 지나가지 못함. 깃발에 계속막히면 일정범위를 강제로 밝힘(폭발). Virus가 활성화 시킨 자리에서 Virus 1개 생성(감염)
